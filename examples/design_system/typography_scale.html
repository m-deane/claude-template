Usage Rules by Size
Decision guide for choosing the right size token on each slide element.
When to use each size token
--text-hero
72px
Big-number slides (stats, KPIs), opening impact statements, closing callout numbers
More than 2 instances per slide; mixed with body copy on the same slide
--text-title
44px
Presentation title (title slide), section divider headings, CTA closing statements
Slide assertion titles (too large — use --text-heading instead); multiple lines at this size
--text-heading
32px
Slide assertion titles (full sentence assertions per slide), section sub-headings, card headlines
Body content (too large for multi-sentence copy); labels on charts (use caption size)
--text-body
24px
Supporting statements (2 sentences max), sub-title beneath slide assertion, speaker-visible key points
Paragraphs longer than 2 lines (split the slide); chart axis labels; captions; footnotes
--text-caption
18px
Figure captions, data source citations, chart axis labels, KPI card labels, slide footnotes
Going below 18px on any element on any slide. 18px is the absolute minimum.
