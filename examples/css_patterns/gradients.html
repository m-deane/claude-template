Usage Notes for reveal.js

How to apply these gradients to reveal.js section elements

Applying gradients to slides

In reveal.js, apply a gradient directly to a <section> element via the data-background-gradient attribute, or use inline style / a CSS class.

/* Option A — data attribute (reveal.js 4+) */ <section data-background-gradient="linear-gradient(135deg, #0f172a 0%, #3b82f6 100%)"> <h1>Title Here</h1> </section> /* Option B — CSS class */ <section class="slide-title"> <h1>Title Here</h1> </section> .slide-title { background: linear-gradient(135deg, #0f172a 0%, #3b82f6 100%) !important; } /* Option C — for geometric overlays, use a wrapper div inside the section */ <section class="slide-section-divider"> <div class="slide-bg-overlay"></div> <h2>Section 2</h2> </section> .slide-section-divider { position: relative; background: linear-gradient(135deg, #1e293b 0%, #0f172a 100%) !important; } .slide-bg-overlay { position: absolute; inset: 0; background-image: linear-gradient(rgba(255,255,255,0.07) 1px, transparent 1px), linear-gradient(90deg, rgba(255,255,255,0.07) 1px, transparent 1px); background-size: 36px 36px; pointer-events: none; }
